4Ô∏è‚É£ How to create a basic server in Node.js?

Use the **http module** to create a simple server.

            
            
const http = require('http');
const server = http.createServer((req, res) => {
    res.writeHead(200, { 'Content-Type': 'text/plain' });
    res.end('Hello, Node.js!');
});
server.listen(3000, () => {
    console.log('Server running on port 3000');
});

5Ô∏è‚É£ What is Express.js?

Express.js is a **fast and minimal web framework** for Node.js. It simplifies routing and middleware management.

const app = express();

    res.send('Hello, Express!');
});

app.listen(3000, () => {
    console.log('Server running on port 3000');
});
